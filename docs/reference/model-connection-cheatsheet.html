模型接入速查表

适用于当前“公众号写作 Agent”的 OpenAI-compatible 接口。

三个必填值

字段含义常见错误
AGENT_BASE_URLAPI 根地址，通常以 /v1 结尾不要再加 /chat/completions
AGENT_API_KEY供应商控制台创建的服务端密钥不要写入网页、聊天或 Git
AGENT_MODEL供应商文档中的精确模型 ID展示名称不一定等于 API 模型 ID

.env.local

AGENT_PROVIDER_MODE=openai-compatible
AGENT_BASE_URL=https://你的服务地址/v1
AGENT_API_KEY=你的本机密钥
AGENT_MODEL=供应商提供的模型ID
AGENT_TIMEOUT_MS=45000
应用会自动在 Base URL 后附加 /chat/completions。若供应商文档给的是完整地址，请只保留它前面的根地址。

成功信号

✓ 健康检查显示 providerMode: openai-compatible
✓ 新建项目后，第一条 Agent 追问来自真实模型
✓ 终端和浏览器没有 401、404、429 或超时提示

错误定位

401密钥无效、过期，或供应商不是 Bearer 鉴权
404Base URL 路径错误，常见于重复填写 /chat/completions
429账户余额、并发或速率配额不足
超时网络不可达、模型过慢，或超时时间过短
结构校验失败模型没有稳定返回 JSON；换用指令遵循更好的模型
